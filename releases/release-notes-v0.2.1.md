## CardieTalkie v0.2.1

This release keeps protocol v2 and is not compatible with the old pre-v0.2.0 firmware.

Highlights:
- refined local PTT start/stop cue behavior on both devices
- start and stop cue levels tuned separately for better balance
- start cue kept audible on the transmitting device
- universal firmware image for both Cardputer and Cardputer ADV
